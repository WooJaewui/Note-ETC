# 인터넷의 주소 체계


1. IP 주소 ( Address )

- 인터넷에 연결된 컴퓨터의 고유한 주소로, 4개의 필드로 끊어서 (.)으로 구분.

- 대한민국은 KRNIC에서 발급.

- IPv6 : IPv4의 32비트 주소 체계로는 전 세계의 주소를 할당하기 어렵기 때문에, 128비트의 주소 체계를 만듬.


2. 도메인 네임 ( Domain name )

- 숫자로 구성되어 있는 IP 어드레스를 문자로 표현한 것.

- [호스트 컴퓨터명].[소속 기관명].[소속 기관의 종류].[소속 국가]


3. DNS ( Domain Name System )

- Domain Name을 IP Address으로 변환해 주는 컴퓨터 체계.


4. URL ( Uniform Resource Locator )

- 인터넷에서 정보의 위치를 알려 주는 표준 주소 체계.

- 인터넷의 정보에 대한 접근 방법, 위치, 파일명 등으로 구성.



# 인트라넷과 엑스트라넷.


1. 인트라넷 : 인터넷의 기술을 기업 내 정보 시스템에 적용한 것.

2. 엑스트라넷 : 기업과 기업 간에 인트라넷을 서로 연결한 것.



# 인터넷 관련 용어.


1. SMTP ( Simple Mail Transfer Protocal ) : 메일 송신 프로토콜.

2. IPTV : 초고속 인터넷을 통해 TV방송 서비스로 시청자가 원하는 시간대에 프로그램을 볼 수 있는 시스템.

3. FTTH ( Fiber To The Home ) : 광섬유를 집 안 까지 설치하는 초고속 인터넷 방식.
