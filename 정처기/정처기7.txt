# 데이터 통신 시스템.


1. 데이터 전송계.

- 단말장치 DTE ( Data Terminal Equipment ) : 데이터의 입출력 및 송수신을 담당하는 장치.

- 데이터 회선 종단 장치 DCE ( Date Circruit-terminating Equipment ) : 신호 변환 장치.

- 통신 제어 장치 CCU ( Communication Control Unit ) : 데이터의 처리를 제어.


2. 데이터 처리계.

- 중앙처리장치 CPU ( Central Processing Unit ) : 연산 장치, 제어 장치, 기억 장치.

- 주변 장치 : 입. 출력 장치, 보조 기억 장치 등.




# 정보 통신의 이용 형태.


1. 온라인 처리 시스템.

- 하나의 중앙 처리 장치에 통신 회선을 통하여 여러 개의 입출력 장치를 항상 연결해서 자료를 처리하는 방식.


2. 오프라인 처리 시스템.

- 중앙 처리 장치와 입출력 장치가 통신 회선으로 연결되지 않고 처리하는 방식.


3. 정보 통신의 처리 방식.

- 일괄처리 ( Batch Processing ) : 한꺼번에 처리하는 방식.

- 실시간 처리 ( Real Time Processing ) : 데이터가 발생하는 즉시 바로 처리하는 방식.

- 시분할 처리 ( Time Sharing System ) : CPU 사용 시간을 일정 시간 단위로 분할하여 처리하는 방식.

- 분산 처리 ( Distributed Processing ) : 데이터를 기능적, 지역적으로 분산하여 처리하는 방식.
