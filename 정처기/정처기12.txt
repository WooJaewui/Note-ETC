# OSI 참조 모델.


Open System Interconnection의 약자로, 개방형 시스템 간의 상호 접속을 위한 참조 모델이다.


1. OSI 참조 모델의 기본 요소.

- 개방형 시스템 Open System : 응용 프로세스 간 원활한 통신을 가능하게 함.

- 응용 개체 Application Entity : 으용 프로그램 같이 실제 정보 처리 및 교환을 수행하는 과정을 모델화.

- 접속 Connection : 응용 개체 간에 논리적인 통신 회선을 의미.

- 물리 매체 Pysical Media : 전기적인 통신 매체.


2. OSI 참조 모델 7계층


하위층

1계층 - 물리계층 : 매체 접근에 따른 기계적, 전기적, 물리적 절차를 규정.

2계층 - 데이터링크계층 : 인접 개방형 시스템 간의 정보 전송 및 오류 제어.

3계층 - 네트워크계층 : 정보교환, 중계 기능, 유통 제어 등.


상위층

4계층 - 전송계층 : 송수신 시스템 간의 논리적 안정 및 균등한 서비스 제공.

5계층 - 세션계층 : 사용자와 전송 계층 간의 인터페이스를 위한 연결.

6계층 - 표현계층 : 정보의 형식 설정 및 부호 교환, 암호화, 해독, 압축 등.

7계층 - 응용계층 : 응용 프로세스 간의 정보 교환 및 전자 사서함, 파일 전송 등.



# 정보 통신망의 구성 형태와 특징.


1. 스타형 : 중앙의 컴퓨터와 단말기들이 1:1로 연결되어 있는 형태. 네트워크 구성의 가장 기본적인 형태.


2. 트리형 : 분산 처리 시스템이 가능하고 통신 선로가 가장 짧음. 제어가 간단하며 관리가 쉬움.


3. 링 ( 루프 ) 형 : 서로 이하는 것 끼리만 연결된 상태. ( LAN에서 가장 많이 사용 )


4. 버스형 : 한 통신 회선에 여러 대의 단말기가 접속되어 있는 형태, 노드의 독립성 보장.


5. 망( Mesh )형 : 모든 단말기와 단말기들을 통신 회선으로 연결시킨 형태.

통신 회선의 링크 수 : n(n-1)/2



# 정보 교환망의 분류.


1. 회선 교환망 ( Circuit Switching Network )

- 컴퓨터와 단말기, 컴퓨터 간 통신 회선을 설정하여 데이터를 교환하는 방식.

- Point to Point 방식으로, 통신 회선을 고정적으로 할당함.

- 실시간 대화용 방식에 응용 가능.

- 음성 전화 시스템과 같이 연속적인 흐름을 유지하는 데이터 교환에 적잘함.


2. 패킷 교환망 ( Packet Switching Network )

- 전송할 데이터를 패킷이라는 일정한 길이로 구분하여 보내는 방식.

- 송신측에서 패킷으로 구분하고, 수신측에서 다시 조립하는 방식.

- 공중 교환 데이터망 ( PSDN )에서 대부분 사용하며, PAD가 필요하다.


3. 메시지 교환망 ( Message Switching Network )

- 수신한 데이터를 중앙에서 축적, 처리하는 방식 ( 축적 교환식 )

- 오류 제어, 코드 변환, 전송하는 메시지 길이에 제약이 없음.

- 메시지의 우선순위 부여 가능.

- 응답 시간이 느리므로 대화형으로는 응용이 불가함.



# 정보 통신망의 종류.


1. LAN ( Local Area Network )

- 광대역 전송 매체의 사용으로 고속 통신이 가능하며 에러율이 낮음.

- 채널의 공동 사용으로 인해서 경로 선택 ( Routing )이 필요 없음.

- 자원의 공유와 데이터의 일관성.


2. 광역 통신망 ( WAN : Wide Area Network )

- 다국적 기업 또는 상호 유대 관계가 있는 기관 간을 LAN으로 연결시킨 망.

- 폐쇄적인 양상이 있기 때문에 전문성과 안정성이 높다.


3. 종합 정보 통신망 ( ISDN : Integrated Services Digital Network )

- 음성 및 데이터나 화상 등 다양한 형태의 통신 서비스를 하나의 통신망으로 제공하는 종합 정보 통신망.


4. 비대칭 디지털 가입자 회선 ( ADSL : Asymmetric Digital Subscriber Line )

- 가정과 회사에 설치되어 있는 전화 회선을 통해 높은 대역폭으로 디지털 정보를 전송하는 기술.

- 같은 회선에 디지털 정보뿐 아니라 아날로그 정보도 동시에 수용 가능.
