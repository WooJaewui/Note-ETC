# 유선 선로 ( 유도 매체 )


1. 나선 open wire

- 가장 최초로 쓰여진 통신 선로로, 전신주를 이용.

- 현재는 거의 사용되지 않음.


2. 트위스티드 페어 케이블 Twisted Pair Cable.

- 2선의 구리선이 서로 감겨 있는 형태의 케이블.

- 설치가 간단하고 저렴하여 가정용 전화기, 개인용 컴퓨터 연결 등에 사용됨.

- 대역폭이 넓지 못하고 고속 전송이 제한적.


3. 동축 케이블 Coaxial Cable. ***

- 동일한 중심축에 플라스틱 절연체를 씌우고 그 위에 그물 모양의 구리망을 감싼 전송 매체.

- 고주파 전송, 광대역 전송, 장거리 전송에 적합하지만 최근에 광대역 케이블로 대체되어 가고 있음.

- TV급 전선, 유선 방송, 근거리 네트워크 등에 사용.


4. 광섬유 케이블 Optical Fiber Cable.*****

- 규소를 주재료로 하며 빛의 전반사 현상을 이용한다.

- 온도 변화에 안정적이며 신뢰성이 높고 에러 발생률이 가장 적음.

- 대륙 간 통신 매체로도 사용된다.

- 코어와 클래딩으로 구성된다.

- 전력 유도나 전자 유도에 영향을 받지 않으므로 잡음이나 누화가 거의 없고 신호 감쇠가 적음.

- 광대역 전송, 작은 크기와 무게, 적은 감쇠도, 보다 넓은 리피터 간격.

- 빛을 이용하여 전송하므로 보안성이 뛰어남.

- 광통신의 3요소 : 발광기, 수광기, 광성유 케이블.

- 광섬유의 굴절률 및 전파 모드 : 1. 단일 모드 : 코어를 직진하는 전파 모드의 수가 하나뿐인 광섬유.

2. 계단형 다중모드 : 클래딩과 코어의 굴절률이 계단형으로 변화.

3. 언덕형 다중모드 : 코어의 굴절률이 완만하게 저하됨.

4. 삼각형 다중모드 : 계단형 장점 + 언덕형 장점.

- 이용분야 : 장거리 중계선, 근거리 통신망, 전화국 간의 중계선.




# 무선 선로 ( 비유도 매체 )


1. 지상 마이크로파 Terrestrial Microwave

- 접시형 안테나를 사용.

- 대용량, 고속 통신이 가능하고 외부의 영향을 적게 받음.

- TV, PCS( 개인용 휴대 통신 서비스 ), 무선 LAN 등.


2. 위성 마이크로파 Satellite Microwave

- 통신 위성을 이용하여 정보를 전송.

- 전파 지연 시간이 발생하며 기후의 영향을 받음.

- 통신 용량이 대용량이며, 국가 간의 원거리 통신에 주로 이용.

- TV, 국제전화, 전신, 사설 기업망 등.


3. 무선 주파수 Radio Frequency.

- 다방향성으로 접시형 안테나가 필요 없음.

- VHF와 UHF 대역의 FM 라디오, UHF, VHF TV 등.


4. 최초의 통신 위성과 위치 및 무선 주파수.

- 통신 위성의 위치 : 적도 위 약 36,000km 상공.

- Telstar( 텔스타 ) : 세계 최초의 통신 위성 ( 1960년 )

- 무궁화 1호 : 우리나라 최초의 통신 위성 ( 1995년 )

- 무선 주파수 : 사람이 들을 수 있는 9kHz ~ 수천GHz까지의 주파수.




# 전송 성로의 이론.


1. 전기적인 1차 정수.

- 도체 저항 R : 물체의 운동 방향과 반대방향으로 작용하는 힘.

- 인덕턴스 L : 전류 회로에 발생된 자기 유도의 정도를 전류 변화와 관련하여 나타내는 양.

- 정전 용량 C : 도체의 형상이나 주위의 유전체에 의해 정해지는 상수.

- 컨덕턴스 G : 저항이 얼마나 잘 흐르는가를 나타냄.


2. 극소 조건 Distortionless Condition.

- 극소 조건 LG = RC




# 교환회선과 전용회선.


1. 교환 회선.

- 교환기를 통하여 연결된 여러 단말 장치에 대해 송수신 하는 방식.

- 전송할 데이터의 양이 적고 접속할 경우가 많은 경우에 유리.

- 'Dial - up' 회선이라고도 함.


2. 전용 회선.

- 교환기를 사용하지 않음.

- 직접적, 직통적, 고정적으로 연결 ( 점 대 점 )하는 방식.

- 데이터의 양이 많거나 이용 횟수가 많을 경우에 적합.




# 통신 방식에 따른 분류.


1. 단방향 통신 ( Simplex ) : 한 쪽 방향으로만 정보 전송이 가능. ex ) TV, 라디오.


2. 반이중 통신 ( Half Duplex ) : 양쪽 방향으로 통신이 가능하지만 동시에는 불가능. ex ) 무전기.


3. 전이중 통신 ( Full Duplex ) : 양쪽 방향으로 동시에 정보의 전송이 가능. ex ) 전화기.




# 포인트 투 포인트 방식, 멀티 포인트 방식.


1. 포인트 투 포인트 Point to Point.

- 점 대 점 방식.

- 데이터를 송수신하는 2개의 단말 장치를 전용 회선으로 항상 접속해 두는 방식.

- 송수신하는 데이터의 양이 많을 경우 유리.

- 컨텐션 ( Contention ) : 송신 요구를 먼저 한 쪽이 송신권을 먼저 갖는 방식.


2. 멀티 포인트 Multi-Point.

- 다중 점 방식, 멀티 드롭 ( Multi-Drop ) 방식이라고도 함.

- 하나의 회선에 여러 단말 장치를 접속하는 방식.

- 폴링과 셀렉션으로 제어하여 송수신.

- 셀렉션 ( Selection ) : 호스트 컴퓨터가 특정 단말장치에 "준비가 되어 있는지" 라고 묻는 제어 방법.

- 폴링 ( Polling ) : 호스트 컴퓨터가 단말 장치들에게 "보낼 데이터가 있는지" 라고 묻는 제어 방식.




# 회선 제어 절차.


회선 연결 => 링크 확립 => 메세지 전달 => 링크 단절 => 회선 절단.




# 전송 속도.


BPS ( Bit Per Second ) : 1초당 전송되는 비트의 수.

Baud ( 보 ) : 1초당 최단 펄스의 수, 매 초당 신호 변화의 개수, 변조 속도.




# BPS와 Baud의 관계.


한 비트가 하나의 신호를 표현하는 단위 Baud로 쓰이는 경우 BPS와 Baud 속도는 동일함.

2비트, 3비트가 모여서 하나의 신호를 나타내는 경우에는 Baud 속도는 BPS의 1/2, 1/3이 됨.




# 통신 용량 ( 채널 용량 )


- 정보가 오류없이 채널을 통해 보내어질 수 있는 최대 속도를 의미함.

- 단위 시간 동안 한 개의 회선으로 최대로 전송할 수 있는 전송 데이터의 양을 의미한다.

- BPS단위로 표현된다.

- 잡음이 없는 이상적인 채널의 경우 신호대잡음비가 무한대가 되어 전송률을 무한대로 높일 수 있다.

- 주파수 대역폭과 통신 속도는 비례함.

# 정보 전송 부호.


1. Baudot 코드 : 5비트 코드.

2. BCD 코드 : 6비트 코드. Digit : 4, Zone : 2, 대소문자 구별되지 않는다.

3. ASCII 코드 : 7비트 코드, Digit : 4, Zone : 3, 대소문자 구별 가능. 데이터 통신용으로 사용.

# 직렬 전송과 병렬 전송.


1. 직렬 전송 Serial Transmission.

- 하나의 문자를 구성하는 각 비트들이 하나의 전송 선로를 통하여 순차적으로 전송됨.

- 통신 설치 비용이 저렴.

- 원거리 전송에 적합.

- 전송 속도 느림.

- 데이터 통신에서 사용.


2. 병렬 전송 Parallel Transmission.

- 한 문자를 구성하는 각 비트들이 각각의 전송 선로를 통하여 동시에 전송.

- 송수신기가 단순, 전송 거리에 따라 전송로의 비용 증가.

- 전송 속도가 빠름.

- 컴퓨터와 주변 기기 간의 통신에 사용.



# 비동기식 전송과 동기식 전송.


1. 비동기식 전송 Asynchronous Transmission.

- 한 문자 단위 ( 5~8비트 )로 전송하는 방식.

- 스타트 비트와 스톱 비트를 삽입하여 전송. ' 스타트 스탑 전송 ' 이라고도 한다.

- 각 문자 사이에 유휴 시간이 있을 수 있음.

- 2,000bps 이하의 낮은 전송 속도에서 이용.

- 주파수 편이 변조 FSK를 사용.


2. 동기식 전송 Synchronous Transmission.

- 여러 문자를 블록 단위로 전송하는 방식.

- 많은 양을 전송하므로 버퍼 장치가 필요.

- 대부분 통신 프로토콜에서 사용.

- 2,400bps 이상의 고속 전송에 사용.

- 위상 편이 변조 PSK를 사용.

- 문자 동기 방식과 비트 동기 방식으로 구분.

- 문자 동기 방식 : 문자를 이용하여 동기화하는 방식.

- 비트 동기 방식 : 데이터 블록의 앞부분과 끝 부분에 플래그나 특수 비트를 전송하여 동기를 맞추는 방식.




# 변조


디지털 또는 아날로그로 부호화된 신호를 전송 매체가 전송할 수 있도록 특정 주파수와 대역폭을 갖는 신호를 생성하는 과정.


아날로그 신호 : 정현파 ( Sine Wave - 구부렁 )

디지털 신호 : 구형파 ( Square Wave - 사각 )




# 변조 방식.


1. 아날로그 데이터의 아날로그 부호화.

- 진폭 변조 ( AM ) : 반송파 진폭을 변조.

- 주파수 변조 ( FM ) : 반송파의 주파수를 변조.

- 위상 변조 ( PM ) : 반송파의 위상을 변조.


2. 아날로그 데이터의 디지털 부호화.

- PCM ( Pulse Code Modulation 방식 )

: - 아날로그 신호를 디지털 펄스로 변환하여 전송하고 수신측에서 이를 다시 본래의 아날로그 신호로 환원시킴.

- 음성 데이터를 디지털 데이터로 표현할 때 사용하는 방식.

- 누화, 잡음, 진폭의 변동에 강하고 전송 레벨에 변동이 없으나, 점유 주파수 대역폭이 큼.

- 시분할 다중화 방식에 사용.

- PCM 변조 과정.

: 아날로그 데이터 -> 표본화 -> 양자화 -> 부호화 -> 복호화 -> 여과 -> 아날로그 데이터.

- PCM - 24.

: - 하나의 전송로를 24개의 채널로 분할하여 전송.

- 음성 최대 주파수 : 4,000Hz.

- 표본화 주파수 : 8,000Hz ( 표본화 공식에 의해 최대 주파수의 2배 )


3. 디지털 데이터의 아날로그 부호화.

- 진폭 편이 복조 ( ASK ) : 300bps 저속 모뎀에서 사용.

- 주파수 편이 변조 ( FSK ) : 1,200bps 이하의 비동기식 모뎀에서 사용.

- 위상 편이 변조 ( PSK ) : 2,400 ~ 4,800bps의 중속 모뎀에서 사용.

- 진폭 위상 변조 ( QAM ) : 4,800bps 이상의 고속 모뎀에서 사용.


4. 디지털 데이터의 디지털 부호화. ( 베이스밴드 전송, 기저 대역 전송 )

- 품질은 우수하나, 디지털 펄스 형태로 전송되므로 정보 손실이 크고 장거리 전송에 부적절함.

- 단류 방식 : 비트가 0일때 0전압, 비트가 1일때 ( + ), ( - ).

- 복류 방식 : 비트가 0일때 ( - )전압, 비트가 1일떄 ( + ) 전압.

- RZ ( Return to Zero ) : 비트 펄스 사이에 0을 일정 시간 유지한 다음 전송하는 방식.

- NRZ ( Non-Return to Zero ) : 0을 유지하지 않고 계속 펄스를 전송하는 방식.

- 단극성 ( Unipolar ) : 비트가 0일때 ( - ) 전압, 비트가 1일때 ( + ) 전압. 동일한 신호일 때 상태 변화 없음.

- 양극성 ( Bipolar ) : 비트가 0 일때 0, 비트가 1일때 ( + ), ( - )을 교대로 사용.




# 전송 제어 문자.


- STX ( Start of TeXt ) : 본문의 개시 및 정보 메시지 헤더의 종료를 표시.

- SOH ( Sratf Of Heading ) : 헤더의 첫 번째 문자로 사용.

- ENQ ( EnQuiry ) : 데이터 링크의 설정 및 응답을 요구.

- ACK ( ACKnowledge ) : 긍정 응답.

- NAK ( Negative AcKnowledge ) : 부정 응답.

- ETX ( End of TeXt ) : 본문의 종료.

- EOT ( End Of Transmission ) : 전송의 종료.

- DLE ( Data Link Escape ) : 데이터 전송 제어 기능을 제공하기 위해 사용.

- SYN ( SYNchronous idle ) : 동기를 취하거나 동기를 유지하기 위해 사용.

- ETB ( End of Transmission Block ) : 전송 블록의 종료를 표시.




# 에러 검출 부호 방식.


1. 패리티 Parity 검사.

- 한 블록의 데이터 끝에 패리티 비트를 추가하는 에러 검출 방식.

- 오류 검출만 가능.

- 홀수 패리티 검사 : 1의 개수가 홀수 개인지를 검사.

- 짝수 패리티 검사 : 1의 개수가 짝수 개인지를 검사.


2. 순환 잉여 검사 ( CRC : Cyclic Redundaancy Check )

- 다항식 코드를 사용하여 오류를 검사하는 방식으로, 블록마다 검사용 코드 FCS를 부가시켜 전송하는 방식.

- 데이터를 연속해서 전송하는 경우 집단 에러 검출을 위해 사용.

- 동기식 전송에서 주로 사용하며 여러 방식 중 신뢰성이 가장 우수함.




# 에러 복구 기법.


1. 자동 재전송 방식 ( ARQ : Automatic Repeat reQuest )

- 에러 검출 후 송신측에게 에러가 발생한 데이터 블록을 다시 전송해 주도록 요청함으로 써 에러를 정정하는 방식.

- Stop & Wait ARQ : 송신측에서 ACK or NAK을 보내올때까지 기다렸다가, NAK 발생시 블락을 재전송.

- Go-back-N ARQ : 에러가 발생한 블록 이후의 모든 블록을 재전송하는 방식.

- 선택적 ARQ : 오류가 발생한 블록만을 재전송하는 방식.

- 적응적( Adaptive ) ARQ : 블록의 길이를 채널의 상태에 따라 동적으로 변경하는 방식.

가장 이상적이지만, 제어 회로가 복잡하여 거의 사용되지 않음.


2. 해밍 코드 ( Hamming Code )

- 단일 비트 에러 검출 및 교정이 가능한 코드.

- FEC ( Forward Error Correction ) 전진 에러 수정 방식의 코드.




# 전송 에러.


1. 잡음 ( Noise )

- 전송로 상에서 전송 신호에 유입되는 불필요한 신호.

- 백색 잡음 ( White Noise ) : 열 운동에 의해 생기는 잡음.

- 충격성 잡음 ( Impulse Noise ) : 제거될 수 없는 잡음. 외부의 충격, 통신 시스템의 결함 등의 의해 발생.

- 누화 잡음 ( Cross Talk Noise ) : 인접 선로의 상호 간섭에 의해 발생. ex ) 전화의 혼선.

- 위상 지터 잡음 ( Phase Jitter Noise ) : 재생 펄스가 위치에서 벗어나 위상이 변하는 현상.


2. 감쇠 ( Attenuation )

- 거리가 멀어질수록 전자적 신호의 세기가 점차적으로 약해지는 현상.

- 아날로그 신호는 증폭기를 사용하여 신호를 회복하고, 디지털 신호는 리피터로 비트 정보를 복원.


3. 왜곡 ( Distortion )

- 주파수 성분들의 특성에 의해 원래 신호가 다른 형태로 일그러지는 현상.

# 정보 단말 장치의 기능.


정보 단말 장치는 신호를 송수신하는 측의 한 쪽 끝에 있는 장치이며, DTE( Data Terminal Equipment )이다.


단말 장치 기능 3가지.


1. 입출력 기능.

2. 전송 제어 기능.

3. 기억 기능.




# 단말 장치 사이의 접속 규격.


1. EIA : RS-232C = 2번핀 : 송신 데이터.

3번핀 : 수신 데이터.

4번핀 : 송신 요구.

5번핀 : 송신 준비 완료.

6번핀 : 모뎀 준비 완료.


2. ITU - T : V.24 = V시리즈 : 아날로그 통신에 대한 권고안.

X 시리즈 : 디지털 통신에 대한 권고안.




# DCE 신호 변환 장치


송신측과 수신측에 알맞은 형태로 변환해 주는 장치이다.

모뎀, DSU, 코덱, 전화 등.




# 모뎀 ( 변복조기 : MODEM )


변조는 디지털 신호를 아날로그 신호로, 복조는 아날로그 신호를 디지털 신호로 변환하는 것.


1. 멀티 포트 : 고속 동기식 모뎀과 시분할 다중화 장비가 혼합된 모뎀.

2. 멀티 포인트 모뎀 : 고속 폴링을 적용한 모뎀.

3. 널 모뎀 : 모뎀 없이 정보를 교환할 수 있는 전송 케이블.

4. 다이얼 - 업 모뎀 : 전화선에 연결되는 모뎀.

5. 음향 결합기 : 전화기의 송수화기를 연결하는 모뎀.




# DSU ( Digital Service Unit )


디지털 방식으로 전송하는 장비이다.

송식측에서부터 수신측까지 디지털 전송이 이루어짐.




# 다중화기 ( Multiplexer )


다수의 단말기들이 각 신호를 하나의 통신 회선에 결합된 형태로 전송하면 수신측에서 분리하여 입출력하는 장비.


1. 주파수 분할 다중화기 ( FDM : Frequency Division Multiplexer )

- 높은 주파수 대역폭을 각기 다른 주파수로 변조해 전송.

- 1,200bps 이하의 저속 아날로그 전송, 비동기 전송에 이용.

- 가드 밴드를 주어야 하며, 그로 인해 대역폭이 낭비됨.

- 변복조기가 필요 없음, 가격 저렴.

- 멀티 포인트 방식에 주로 사용.

- 라디오, TV, 통신 위성, CATV 등에 이용.


2. 시분할 다중화기 ( TDM : Time Division Multiplexer )

- 여러 회선의 음성 정보를 작은 시간으로 나누어 고속의 전송로로 보내는 방식, 하나의 회선을 다수 채널로 다중화.

- 디지털 전송에 적합.

- PCM 방식에서 사용. ( 아날로그를 디지털로 변환하여 전송 )

- 고속 전송이 가능.

- 포인트 투 포인트 방식에 주로 사용.

- 동기식, 비동기식 모두 사용 가능.



# 집중화기 ( Concentrator )


여러 개의 입력 회선이 적은 출력 회선을 동적으로 이용하여 저속 장치들이 하나의 고속 회선을 이용.

시스템이 복잡하며, 불규칙한 전송에 적합.

전처리기, 선로 공동 이용기 등.




# 전화기 ( Telephone )


사람의 음성을 전기적인 신호로 바꾸고 다시 그 전기신호를 음성으로 변환하는 기기.


1. MFC 전화기 : - 3X4배열 누름 버튼 중 10 개는 송출, 2개는 기능 버튼.

- 2가지 주파수를 혼합하여 번호를 식별하는 전화기.

- 다이얼 방식에 비하여 정확한 접속.




# 코드 분할 다중 접속 ( CDMA : Code Division Multiple Access )


통화시 음성 신호를 비트 단위로 분할해 코드화한 후 이 신호를 통신 주파수 대역에 삽입하는 방식.

페이딩 제거 현상 및 핸드오프 기능이 우수하고, 사용되는 주파수 신호가 멀리 전달되며 잡음이 적음.




# 도플러 효과 ( Doppler Effect )


이동 통신의 전파 특성으로 소리의 고저가 고유 본래의 음과 다르게 들리는 현상.
