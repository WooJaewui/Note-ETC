# 유선 선로 ( 유도 매체 )


1. 동축 케이블 ( Coaxial Cable )

- 동일한 중심축에 플라스틱 절연체를 씌우고 그 위에 그물 모양의 구리망을 감싼 전송 매체.

- 고주파 전송, 광대역 전송, 장거리 전송에 적합하지만 최근에 광대역 케이블로 대체되어 가고 있음.


2. 광섬유 케이블 ( Optical Fiber Cable )

- 규소를 주재료로 하며 '빛의 전반사' 현상을 이용.

- 코어와 클래딩으로 구성되며, 누화 감쇠가 거의 없다.

- 보안성이 뛰어남.

- 광통신의 3요소 : 1. 발광기. 2. 수광기. 3. 광섬유 케이블.

- 광섬유의 굴절률 및 전파 모드 : 1. 단일모드 SM. 2. 계단형 다중모드 SIMMF. 3. 언덕형 다중 모드 GIMMF.

4. 삼각형 다중 모드 TIMMF.




# 무선 선로 ( 비유도 매체 )


1. 지상마이크로파 ( Terrestrial Microwave )


2. 위성 마이크로파 ( Satellite Microwave )

- 통신 위성을 이용한 정보 전송.

- 전파 지연 시간이 발생하며 기후의 영향을 받음.


3. 무선 주파수 ( Radio Frequency )


4. 최초의 통신 위성과 위치 및 무선 주파수.

- 통신 위성의 위치 : 적도 위 약 36,000km 상공.

- Telstar( 텔스타 ) : 세계 최초의 통신 위성 ( 1960년 )

- 무궁화 1호 : 우리나라 최초의 통신 위성 ( 1995년 )

- 무선 주파수 : 사람이 들을 수 있는 9kHz ~ 수천GHz까지의 주파수.




# 전송 선로의 이론.


1. 전기적인 1차 정수.

- 도체 저항 ( R )

- 인덕턴스 ( L )

- 정전 용량 ( C )

- 컨덕턴스 ( G ).


2. 극소 조건.

- LG = RC.



# 교환 회선과 전용 회선.


1. 교환 회선.

- 교환기를 통하여 연결된 여러 단말 장치에 대한 송수신 하는 방식.

- 전송할 데이터의 양이 적고 접속할 경우가 많은 경우에 유리.


2. 전용 회선.

- 교환기를 사용하지 않음.

- 직접적, 직통적, 고정적으로 연결 ( 점 대 점 ) 하는 방식.

- 데이터의 양이 많거나 이용 횟수가 많을 경우에 적합.




# 통신 방식에 따른 분류.


1. 단방향 통신 : 한 쪽 방향으로만 정보 전송이 가능. ex ) TV, 라디오.


2. 반이중 통신 : 양쪽 방향으로 통신이 가능하지만 동시에는 불가능. ex ) 무전기.


3. 전이중 통신 : 양쪽 방향으로 동시에 정보의 전송이 가능. ex ) 전화기.



# 포인트 투 포인트 & 멀티 포인트.


1. 포인트 투 포인트

- 데이터를 송수신하는 2개의 단말 장치를 전용 회선으로 항상 접속해 두는 방식.

- 송수신하는 데이터의 양이 많을 경우 유리.

- 컨텐션으로 제어. ( 송신 요구를 먼저 한 쪽이 송신권을 먼저 갖는 방식 )


2. 멀티 포인트 방식.

- 멀티 드롭 방식.

- 하나의 회선에 여러 단말 장치를 접속하는 방식.

- 폴링과 셀렉션으로 제어하여 송수신.

# 전송 속도.


1. BPS : 1초당 전송되는 비트의 수.


2. Baud : 1초당 최단 펄스의 수.


3. 최단 Baud 구하는 방법 : baud = 1/time.


4. 위상에 따른 baud와 bps.

- 2위상 : bps = baud.

- 4위상 : bps = baud x 2.

- 8위상 : bps = baud x 3.
