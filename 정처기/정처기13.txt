출처 입력


# 입출력 기능.


1. 입력장치.

- 키보드, 마우스, OMR, OCR, MICR, 디지타이저, 스캐너, CIM.


2. 출력장치.

- 영상 표시 장치, X - Y 플로터, COM, 프린터.




# 입출력 채널 ( I / O Channel )


1. 채널의 정의 : CPU의 처리 효율을 높이고 데이터의 입출력을 빠르게 할 수 있게 만든 입출력 전용 처리기.

입출력 장치와 주기억 장치 사이의 속도 차이를 위한 장치. ( 자체 메모리 없음 )

CPU의 간섭 없이 입출력을 수행하며, 작업 완료시 인터럽트로 알림.


2. 채널의 종류 : 셀렉터 채널 - 한 번에 한 개의 장치를 선택하여 동작.

멀티플렉서 채널 - 동시에 여러 개의 장치를 선택하여 동작.

블록 멀티 플렉서 채널 - 셀렉터와 멀티플렉서의 장점만을 모아 만든 채널이다.




# DMA ( Direct Memory Access )


CPU의 간섭 없이 주기억 장치와 입출력 장치 사이에서 직접 전송이 이루어지는 방법.

CPU와 주변 장치 간의 속도 차이를 줄일 수 있음.

고속으로 대량의 데이터를 전송.

Cycle Stealing 방식 이용. ( 우선순위가 높은 채널이 인터럽트를 발생시켜 주기억 장치를 사용 하는 것 )




# I / O Channel , DMA


I / O Channel : 말그대로 입출력을 하는 것이다.

DMA : 주기억장치와 주변장치 사이에 data 전송과 관련이 있는 것이다.




# 인터럽트


컴퓨터에서 정상적인 프로그램을 처리하고 있는 도중 특수한 상태가 발생했을 때 현재 실행하고 있는 프로그램을

일시 중지하고, 특수한 상태를 처리한 후 다시 원래의 프로그램으로 복귀하여 처리하는 것을 의미.




# 인터럽트의 종류.


1. 하드웨어 인터럽트

- 정전 인터럽트 : 정전시 발생.

- 기계고장 인터럽트 : 기계 고장시 발생.

- 외부 인터럽트 : 타이머 종료, 오퍼레이터의 콘솔 버튼 조작.

- 입출력 인터럽트 : 데이터의 I / O 종료, 오류.


2. 소프트웨어 인터럽트.

- 프로그램 인터럽트 : 무한 루프나 0으로 나누는 연산 등 프로그램 명령시 사용법이나 지정법에 잘못이 있을 때.

- 슈퍼바이저 콜( SVC ) 인터럽트 : 감시자의 호출, SVC 명령 실행.




# 주기억 장치.


1. ROM ( Read Only Memory )

- 기억된 내용의 읽기만 가능한 장치.

- 전원의 공급이 끊어져도 그 내용을 기억하는 비휘발성 기억 장치.

- ROM BIOS 등을 내장.


2. RAM ( Random Access memory )

- 읽기 / 쓰기가 가능한 기억 장치.

- 전원의 공급이 끊어지면 그 내용을 잃어버리는 휘발성 메모리.

- RAM의 종류 및 특성 : SRAM - 소비 전력이 많고 속도가 빠름. 캐시메모리로 사용

DRAM - Refresh 필요하고, 속도가 SRAM보다 느림. 주기억 장치로 사용.




# 보조기억장치.


1. 자기 디스크 ( Magnetic DisK )

- 기억 용량이 크고, 처리 속도가 빠르며 DASD로 순차 비순차 처리가 모두 가능하다.

- 10장의 디스크가 있으면 : 18면 사용 가능. ( 맨 위와 맨 아래면은 사용 불가 )


2. 자기 테이프 ( Magnetic Tape )

- SASD로 순차 처리만 가능하다.

- 데이터를 한꺼번에 모아서 처리하는 일괄 처리 방식을 사용.

- 백업용으로 많이 사용.




# 기타 기억장치.


1. 캐시기억장치.

- CPU와 주기억 장치 사이에 있는 고속의 버퍼 메모리.

- 자주 참조되는 데이터나 프로그램을 메모리에 저장.


2. 가상 기억 장치 ( Virtual Memory )

- 기억 장소를 주기억 장치의 용량으로 제한하지 않고 보조 기억 장치까지 확대하여 사용.

- 공간의 확대에 목적이 있음 ( 처리 속도가 향상되는 것은 아님 )

- 자기 디스크를 많이 사용.

- 일정하지 않은 세그먼트나 일정한 페이지로 분할해서 사용하는 페이징 기법을 사용함.


3. Firmware

- 소프트웨어이지만 하드웨어에 저장되어 반영구적으로 사용.

- BIOS와 같은 프로그램을 ROM에 고정시켜 하드웨어화한 것을 의미함.
